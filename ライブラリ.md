## divmod
`divmod(a,b)`でaをbで割った商と余りのタプルを返す
```py
q,r = divmod(20,7)
# q = 2, r = 6
```
## 値を入れ替える
```py
x, y = y, x
```
## リスト
* 順番のリストを作成
```py
a = list(range(5))
# a = [0,1,2,3,4]
```
* 入力からリストを作成
```py
a = list(map(int, input().split()))
# 入力：4 2 5 1
# a = [4,2,5,1]
```
* 要素追加・削除
```py
a = []
# 追加
a.append(1) # a = [1]
a.append(5) # a = [1,5]
# 削除
a.pop() # a = [1]
```
